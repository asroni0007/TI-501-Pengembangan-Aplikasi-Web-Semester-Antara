# Panduan Upload Pertemuan 5 ke GitHub

Unggah folder ini ke repository utama dengan nama:

```text
05_javascript-dom-event-fetch
```

Pastikan struktur folder tetap seperti berikut:

```text
05_javascript-dom-event-fetch/
├── README.md
├── PANDUAN_UPLOAD.md
├── assets/
├── demo-opsional/
├── materi/
├── panduan/
├── src/
└── tugas/
```

## Catatan

- `src/` adalah source code latihan mahasiswa.
- `demo-opsional/` dapat digunakan dosen untuk demonstrasi.
- Materi ini belum menggunakan Node.js atau backend. Node.js dimulai pada Pertemuan 6.
