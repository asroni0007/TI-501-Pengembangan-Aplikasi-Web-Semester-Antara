# Rubrik Penilaian Pertemuan 5

**Bobot aktivitas: 4%**

| Aspek | Bobot | Kriteria |
|---|---:|---|
| Struktur HTML dan CSS | 15% | Halaman rapi, responsif, dan tetap menggunakan struktur semantik. |
| DOM selector dan manipulasi | 20% | Elemen dipilih dan diperbarui dengan tepat. |
| Event handling | 20% | Event berjalan benar, form tidak reload, dan alur interaksi jelas. |
| Validasi input | 15% | Ada validasi minimal dan pesan kesalahan mudah dipahami. |
| Fetch API | 20% | Data eksternal/lokal berhasil diambil dan dimanfaatkan. |
| Kerapian kode dan refleksi | 10% | Kode mudah dibaca, tidak ada error console, dan refleksi jelas. |
