# Instruksi Tugas Pertemuan 5

## Judul
Halaman Interaktif Asisten Caption UMKM dengan JavaScript DOM, Event, dan Fetch API

## Output
Mahasiswa mengembangkan halaman web sederhana yang memiliki:

1. Form input data produk UMKM.
2. Validasi input minimal menggunakan JavaScript.
3. Event handling pada tombol atau form submit.
4. Manipulasi DOM untuk menampilkan hasil.
5. Fetch API untuk mengambil contoh data dari file JSON lokal atau API publik.
6. Desain responsif dasar dari Pertemuan 3.

## Ketentuan Teknis

- Gunakan HTML, CSS, dan JavaScript murni.
- Tidak perlu Node.js atau backend.
- Jalankan menggunakan Live Server agar `fetch()` ke file JSON lokal dapat berjalan.
- Simpan file utama di folder `src/`.

## Pengumpulan

- Folder proyek dalam format ZIP, atau tautan repository GitHub.
- Nama file: `NIM_Nama_Pertemuan05.zip`.
- Sertakan screenshot hasil halaman dan file refleksi singkat.
