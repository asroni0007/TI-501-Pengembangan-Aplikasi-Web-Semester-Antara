# Checklist Interaktivitas Pertemuan 5

Gunakan checklist berikut sebelum mengumpulkan tugas.

- [ ] Form memiliki label yang jelas.
- [ ] JavaScript terhubung ke HTML.
- [ ] Minimal menggunakan `querySelector` atau `getElementById`.
- [ ] Minimal ada satu `addEventListener`.
- [ ] Form tidak melakukan reload halaman saat submit.
- [ ] Ada validasi input dan pesan kesalahan.
- [ ] Hasil caption ditampilkan melalui manipulasi DOM.
- [ ] Fetch API dapat mengambil data contoh.
- [ ] Halaman tetap rapi di layar kecil.
- [ ] Tidak ada error pada browser console.
