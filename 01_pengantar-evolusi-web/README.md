# Pertemuan 1 — Pengantar Pengembangan Aplikasi Web & Evolusi Web 1.0–3.0

**Sub-CPMK 51.1 · Sesi diskusi/ceramah · Tanpa praktikum coding · Tugas individu 5%**

## Capaian Pembelajaran

Setelah mengikuti pertemuan ini, mahasiswa mampu:

1. Menjelaskan pengertian dan karakteristik aplikasi web.
2. Membedakan Web 1.0, Web 2.0, dan Web 3.0.
3. Menjelaskan arsitektur client-server sederhana.
4. Memetakan frontend, backend, API, dan basis data pada aplikasi web modern.
5. Menyusun analisis singkat perkembangan generasi web.

## Materi

- Pengantar pengembangan aplikasi web.
- Evolusi Web 1.0, Web 2.0, dan Web 3.0.
- Arsitektur client-server.
- Gambaran stack semester: HTML, CSS, JavaScript, Node.js, Express.js, MySQL, dan API AI.
- Pengenalan studi kasus semester: **Asisten AI untuk UMKM**.
