# Panduan Unggah Pertemuan 2 ke GitHub

Unggah folder berikut ke root repository:

```text
02_html5-semantik-form/
```

Pastikan struktur akhirnya:

```text
02_html5-semantik-form/
├── PANDUAN_UPLOAD.md
├── README.md
├── assets/
├── demo-opsional/
├── materi/
├── panduan/
├── src/
└── tugas/
```

## Langkah melalui GitHub Web

1. Buka repository mata kuliah.
2. Pilih **Add file → Upload files**.
3. Tarik seluruh folder `02_html5-semantik-form` ke area unggah.
4. Isi commit message: `Menambahkan materi Pertemuan 2 HTML5 semantik`.
5. Klik **Commit changes**.

## Pemeriksaan Setelah Unggah

- README tampil pada folder Pertemuan 2.
- File PPT dan DOCX dapat diunduh.
- `src/index.html` serta `src/style.css` tersedia.
- Diagram SVG tampil dengan benar.
- Seluruh file tugas tersedia.

Folder `demo-opsional` hanya untuk demonstrasi dosen. Tugas mahasiswa menggunakan berkas pada folder `src`.
