# Instruksi Tugas Pertemuan 2

Buat halaman awal **Asisten AI untuk UMKM** menggunakan HTML5 semantik.

## Ketentuan

- File utama harus bernama `index.html`.
- Gunakan elemen: `header`, `nav`, `main`, `section`, `article`, `aside`, dan `footer`.
- Tambahkan form konsultasi UMKM.
- Setiap input form harus memiliki `label` yang terhubung dengan `id`.
- Gunakan `fieldset` dan `legend` untuk pilihan radio/checkbox.
- Isi checklist aksesibilitas sebelum dikumpulkan.

## Format Pengumpulan

```text
NIM_Nama_Pertemuan02/
├── index.html
├── screenshot.png
└── checklist-aksesibilitas.md
```

Bobot tugas: **3%**.
