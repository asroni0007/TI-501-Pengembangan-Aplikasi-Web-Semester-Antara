# Pertemuan 2 — HTML5 Semantik, Struktur Form, dan Aksesibilitas Dasar

**TI-501 Pengembangan Aplikasi Web — Semester Antara**  
**CPMK52 · Bobot pertemuan 3% · Praktikum HTML tanpa JavaScript**

## Capaian Pembelajaran

Setelah mengikuti pertemuan ini, mahasiswa mampu:

1. Menjelaskan fungsi struktur dasar dokumen HTML5.
2. Menggunakan elemen HTML semantik secara tepat.
3. Membuat form yang memiliki struktur dan label yang benar.
4. Menerapkan aksesibilitas dasar pada halaman web.
5. Mengembangkan fondasi antarmuka proyek **Asisten AI untuk UMKM**.

## Kesesuaian dengan RPS

Materi Pertemuan 2:

- HTML5 semantic elements;
- struktur form;
- aksesibilitas dasar;
- penerapan nilai Al-'Adl dan Rahmah dalam desain inklusif.

## Kelanjutan dari Pertemuan 1

Pertemuan 1 membahas evolusi Web 1.0–3.0 dan arsitektur client-server. Pertemuan 2 mulai membangun lapisan **frontend** dalam bentuk struktur halaman HTML5 untuk studi kasus semester.

## Aktivitas Praktikum

1. Membuka contoh pada folder `src`.
2. Mengidentifikasi struktur `header`, `nav`, `main`, `section`, `article`, `aside`, dan `footer`.
3. Memeriksa hubungan `label` dengan `input` melalui atribut `for` dan `id`.
4. Mengembangkan form konsultasi UMKM.
5. Memeriksa aksesibilitas dasar menggunakan checklist.
6. Mengumpulkan file HTML, CSS, screenshot, dan checklist.

## Menjalankan Contoh

Buka file berikut secara langsung dengan browser:

```text
src/index.html
```

Tidak diperlukan Node.js atau server pada Pertemuan 2.

## Berkas Utama

- `materi/` — slide teori Pertemuan 2.
- `panduan/` — panduan praktikum dalam format DOCX.
- `src/` — contoh kode HTML dan CSS.
- `tugas/` — instruksi, template, checklist, dan rubrik.
- `assets/` — diagram struktur HTML semantik.
- `demo-opsional/` — versi demo mandiri untuk dosen.
