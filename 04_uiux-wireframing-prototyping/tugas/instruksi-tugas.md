# Instruksi Tugas Pertemuan 4

Buat rancangan low-fidelity untuk aplikasi **Asisten AI untuk UMKM**.

## Produk yang dikumpulkan

1. Persona ringkas.
2. Tujuan pengguna.
3. User flow utama.
4. Wireframe 3 halaman:
   - Beranda;
   - Form Produk;
   - Hasil Caption AI.
5. Prototype klik di Figma atau PDF prototype.
6. Refleksi keputusan desain.

## Ketentuan

- Bentuk pengumpulan: tautan Figma atau PDF.
- Nama file: `NIM_Nama_Pertemuan04.pdf`.
- Gunakan template `template-rancangan-uiux.md`.
- Kumpulkan melalui Moodle atau GitHub sesuai arahan dosen.
