# Checklist Usability dan Aksesibilitas

Gunakan checklist berikut sebelum mengumpulkan rancangan.

## Usability

- [ ] Tujuan halaman mudah dipahami.
- [ ] Tombol utama terlihat jelas.
- [ ] Navigasi antarhalaman konsisten.
- [ ] Form tidak meminta data yang tidak perlu.
- [ ] Pesan kesalahan atau arahan perbaikan tersedia.
- [ ] Alur pengguna selesai tanpa langkah berulang yang tidak perlu.

## Aksesibilitas

- [ ] Teks mudah dibaca.
- [ ] Kontras warna cukup jelas.
- [ ] Label input tersedia.
- [ ] Urutan fokus/logika navigasi masuk akal.
- [ ] Ikon penting tetap disertai teks.
- [ ] Prototype tetap dapat dipahami tanpa warna saja.
