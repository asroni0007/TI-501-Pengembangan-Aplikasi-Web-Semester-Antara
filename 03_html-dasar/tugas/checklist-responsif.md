# Checklist Responsif Pertemuan 3

Gunakan checklist ini sebelum mengumpulkan tugas.

- [ ] HTML masih semantik dan tidak berubah menjadi kumpulan `div` saja.
- [ ] File CSS terpisah dan sudah terhubung di `<head>`.
- [ ] Menggunakan `box-sizing: border-box`.
- [ ] Ada penggunaan Flexbox.
- [ ] Ada penggunaan CSS Grid.
- [ ] Ada media query untuk layar kecil.
- [ ] Tampilan mobile tidak melebar ke samping.
- [ ] Teks mudah dibaca pada desktop dan mobile.
- [ ] Tombol memiliki kontras warna yang cukup.
- [ ] Form memiliki label yang sesuai.
- [ ] Fokus input terlihat saat ditekan keyboard.
- [ ] Tidak ada elemen yang saling bertumpuk.
