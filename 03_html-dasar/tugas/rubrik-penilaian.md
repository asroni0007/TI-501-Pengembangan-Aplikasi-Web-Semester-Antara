# Rubrik Penilaian Pertemuan 3

**Bobot pertemuan: 3%**

| Aspek | Bobot | Kriteria |
|---|---:|---|
| Struktur HTML semantik | 15% | Struktur HTML tetap rapi dan sesuai fungsi elemen. |
| Penerapan box model | 15% | Margin, padding, border, dan ukuran elemen terkendali. |
| Penggunaan Flexbox | 20% | Flexbox digunakan tepat untuk navigasi/komponen satu dimensi. |
| Penggunaan CSS Grid | 20% | Grid digunakan tepat untuk layout kartu/section dua dimensi. |
| Responsivitas | 20% | Media query bekerja dan tampilan mobile tidak rusak. |
| Kerapian dan aksesibilitas | 10% | Teks terbaca, warna cukup kontras, form memiliki label, dan fokus terlihat. |

## Skala Nilai

- 85–100: Sangat baik
- 75–84: Baik
- 65–74: Cukup
- <65: Perlu perbaikan
