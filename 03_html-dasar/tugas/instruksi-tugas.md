# Instruksi Tugas Pertemuan 3

## Tugas

Buat tampilan responsif untuk halaman **Asisten AI untuk UMKM** menggunakan HTML dari Pertemuan 2 dan CSS3.

## Ketentuan

1. Gunakan HTML semantik: `header`, `nav`, `main`, `section`, `article`, `form`, dan `footer`.
2. Buat file `style.css` terpisah.
3. Terapkan box model dengan `box-sizing: border-box`.
4. Gunakan Flexbox minimal pada navigasi atau susunan elemen satu baris.
5. Gunakan CSS Grid minimal pada daftar fitur/kartu.
6. Gunakan minimal satu media query untuk layar kecil.
7. Pastikan form tetap memiliki `label` yang jelas.
8. Pastikan warna, ukuran teks, dan jarak antar elemen mudah dibaca.
