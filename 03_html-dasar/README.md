# Pertemuan 3 — CSS3 Layout: Box Model, Flexbox, Grid, dan Responsive Design

**CPMK52 · Praktikum terarah · Bobot 3%**

Pertemuan ini melanjutkan hasil Pertemuan 2 yang sudah membangun struktur HTML semantik. Fokus Pertemuan 3 adalah memberi tampilan dan tata letak responsif menggunakan CSS3.

## Capaian Pembelajaran

Setelah mengikuti pertemuan ini, mahasiswa mampu:

1. Menjelaskan peran CSS dalam memisahkan struktur dan presentasi halaman web.
2. Menerapkan CSS box model: content, padding, border, dan margin.
3. Menggunakan Flexbox untuk menyusun navigasi dan komponen satu dimensi.
4. Menggunakan CSS Grid untuk menyusun layout dua dimensi.
5. Menggunakan media query untuk membuat halaman responsif.
6. Memeriksa keterbacaan, jarak, dan aksesibilitas dasar tampilan web.

## Materi

- CSS syntax, selector, cascade, specificity, dan custom properties.
- Box model dan `box-sizing: border-box`.
- Flexbox: `display: flex`, `gap`, `justify-content`, `align-items`, dan `flex-wrap`.
- CSS Grid: `display: grid`, `grid-template-columns`, `repeat()`, `minmax()`, dan `gap`.
- Media query dan prinsip mobile-first.
- Desain responsif untuk studi kasus **Asisten AI untuk UMKM**.
