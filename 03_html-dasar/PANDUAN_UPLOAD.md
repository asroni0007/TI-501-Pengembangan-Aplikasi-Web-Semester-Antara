# Panduan Upload Pertemuan 3 ke GitHub

Unggah folder ini ke root repository:

`03_css3-layout-responsif/`

Pastikan struktur folder tetap sama agar tautan pada README tidak rusak.

## Langkah Singkat

1. Buka repository GitHub.
2. Pilih **Add file → Upload files**.
3. Unggah seluruh isi folder `03_css3-layout-responsif`.
4. Commit dengan pesan:

```text
Menambahkan materi Pertemuan 3 CSS3 layout responsif
```

## Catatan

Folder `src/` berisi source utama mahasiswa. Folder `demo-opsional/` dapat dipakai dosen untuk demo cepat di browser.
