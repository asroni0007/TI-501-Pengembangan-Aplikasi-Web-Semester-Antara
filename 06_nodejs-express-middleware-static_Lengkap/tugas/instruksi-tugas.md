# Instruksi Tugas Pertemuan 6

Buat server Express sederhana untuk studi kasus **Asisten AI UMKM**.

## Ketentuan

1. Jalankan source pada folder `src`.
2. Pastikan halaman utama tampil di `http://localhost:3000`.
3. Pastikan endpoint berikut aktif:
   - `GET /api/status`
   - `GET /api/produk`
4. Tambahkan minimal satu data produk baru di `data/produk.js`.
5. Ubah tampilan pada `public/style.css` agar halaman lebih rapi.
6. Sertakan screenshot halaman utama, `/api/status`, dan `/api/produk`.
7. Tulis refleksi singkat tentang perbedaan static file biasa dan static file yang dilayani Express.

## Pengumpulan

- Format: ZIP atau link GitHub.
- Nama file: `NIM_Nama_Pertemuan06.zip`.
